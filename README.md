# BarLoop

YouTube 또는 로컬 영상 파일을 BPM 기준 마디 단위로 나누고, 원하는 구간을 반복 재생하는 React 연습 도구입니다.

## 주요 기능

- YouTube URL / 영상 ID 재생
- 로컬 영상 파일 선택 및 드래그 앤 드롭
- 로컬 파일은 서버 전송 없이 `URL.createObjectURL()`로 브라우저에서만 재생
- BPM, 박자 수, 첫 다운비트 기준 마디 자동 분할
- 단일 마디 및 여러 마디 범위 반복
- 직접 지정하는 A–B 시간 구간 반복
- 0.5× ~ 2× 재생 속도 조절
- 타임라인 탐색, 5초 이동, 반복 구간 표시
- 키보드 단축키 및 모바일 대응 UI


## 키보드 단축키

| 키 | 기능 |
|---|---|
| `Space` | 재생 / 일시정지 |
| `←` / `→` | 5초 이전 / 다음 |
| `L` | 반복 ON / OFF |
| `[` | 현재 시간을 A 지점으로 설정 |
| `]` | 현재 시간을 B 지점으로 설정 |

## 로컬 영상 지원

영상 파일은 서버에 업로드하거나 저장하지 않습니다. 브라우저가 파일을 직접 읽기 때문에 새로고침하면 다시 선택해야 합니다.

브라우저 호환성이 가장 높은 형식은 다음과 같습니다.

- 컨테이너: MP4
- 비디오 코덱: H.264
- 오디오 코덱: AAC

MOV, HEVC/H.265 또는 특수 코덱은 운영체제와 브라우저에 따라 재생되지 않을 수 있습니다.

## 반복 정밀도

HTML5 Video와 YouTube IFrame API의 현재 시간을 주기적으로 확인해 구간 끝에서 시작점으로 이동합니다. 음악 연습에는 적합하지만 코덱의 키프레임 구조와 YouTube 플레이어 동작에 따라 수십 ms 수준의 오차가 생길 수 있습니다.
