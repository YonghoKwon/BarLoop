# BarLoop

YouTube, 로컬 영상 또는 음원 파일을 BPM 기준 마디 단위로 나누고 원하는 구간을 반복 재생하는 React 연습 도구입니다.

## 주요 기능

- YouTube URL / 영상 ID 재생
- 로컬 영상·음원 파일 선택 및 드래그 앤 드롭
- 로컬 파일은 서버 전송 없이 `URL.createObjectURL()`로 브라우저에서만 재생
- BPM 입력 시 앞자리 0 자동 제거
- 탭 템포로 BPM 자동 계산
- BPM, 박자 수, 첫 다운비트 기준 마디 자동 분할
- 단일 마디 및 여러 마디 범위 반복
- 직접 지정하는 A–B 시간 구간 반복
- 반복 횟수 표시 및 초기화
- 0.5× ~ 2× 재생 속도 조절
- 5초 이동과 0.1초 미세 이동
- BPM, 박자, 속도, 반복 설정 브라우저 자동 저장
- 키보드 단축키 및 모바일 대응 UI

## 실행

```bash
npm install
npm run dev
```

프로덕션 빌드:

```bash
npm run build
npm run preview
```

## 키보드 단축키

| 키 | 기능 |
|---|---|
| `Space` | 재생 / 일시정지 |
| `←` / `→` | 5초 이전 / 다음 |
| `Shift` + `←` / `→` | 0.1초 이전 / 다음 |
| `L` | 반복 ON / OFF |
| `[` | 현재 시간을 A 지점으로 설정 |
| `]` | 현재 시간을 B 지점으로 설정 |

## 로컬 미디어 지원

로컬 파일은 서버에 업로드하거나 저장하지 않습니다. 브라우저가 파일을 직접 읽기 때문에 새로고침하면 파일을 다시 선택해야 합니다.

권장 형식:

- 영상: MP4(H.264/AAC), WebM
- 음원: MP3, WAV, M4A/AAC, OGG

MOV, HEVC/H.265, FLAC 등은 운영체제와 브라우저에 따라 재생되지 않을 수 있습니다.

## 반복 정밀도

HTMLMediaElement와 YouTube IFrame API의 현재 시간을 주기적으로 확인해 구간 끝에서 시작점으로 이동합니다. 음악 연습에는 적합하지만 코덱 키프레임 구조와 YouTube 플레이어 동작에 따라 수십 ms 수준의 오차가 생길 수 있습니다.
